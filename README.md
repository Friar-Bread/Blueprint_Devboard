# Blueprint_Devboard
This is my custom RP2040 devboard I made while following the Hackclub tutorial.
Below you can see the PCBA image of the circuit.
<img width="833" height="2066" alt="DevboardPCBA_Image" src="https://github.com/user-attachments/assets/3aab339c-48a8-42da-92c1-f7d560262b46" />

I had a lot of fun creating the schematic and routing the PCB for this project.
I went with a funny alien theme because I like the art style and it was easy for me to do.
